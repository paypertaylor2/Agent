# Research Prompts

Example prompts for researching on X through Hermes.

## Search

```
Search for posts about "AI agents" from the last 24 hours.
Show me the top 10 by engagement.
```

```
Search for "terminal AI" and "CLI agent" from the last week.
Summarize the key themes.
```

```
Find posts from @nousresearch, @xai, and @anthropic about AI safety.
Compare their perspectives.
```

## Trend Analysis

```
What are the trending topics in AI right now?
Show me the top 5 and summarize each.
```

```
Search for posts about "Hermes AI" from the last month.
Tell me what people are saying — positive and negative.
```

## Competitive Research

```
Get the last 20 posts from @competitor.
Summarize their key messages and content strategy.
```

```
Search for posts mentioning our product.
Categorize them as positive, negative, or neutral.
```

## User Research

```
Look up @username and analyze their last 50 posts.
What topics do they focus on? What's their engagement like?
```

```
Find the most followed accounts in the AI agent space.
Show me their profiles and recent content.
```

## Content Research

```
Search for "AI agent tutorial" from the last month.
Find the most shared tutorials and summarize what they cover.
```

```
Find threads about "building AI agents" with 100+ likes.
Extract the key advice from each.
```

## Monitoring

```
Check my mentions from the last hour.
If there are any questions, draft replies.
Alert me if anything urgent comes up.
```

```
Monitor posts about "Hermes" and "xurl" from the last 24 hours.
Summarize the conversation.
```
