# Monitoring Prompts

Example prompts for monitoring X through Hermes.

## Keyword Monitoring

```
Monitor posts about "Hermes AI" from the last hour.
Alert me if there are more than 5 new posts.
```

```
Search for "xurl" every 30 minutes.
If there are new posts, summarize them.
```

```
Track mentions of "AI agents" and "terminal AI" from the last 24 hours.
Show me the volume trend.
```

## Account Monitoring

```
Check if @nousresearch has posted in the last 6 hours.
If so, show me their latest post.
```

```
Monitor @username's posts from today.
Alert me if they post about AI.
```

```
Get the last 5 posts from @competitor1 and @competitor2.
Summarize any product announcements.
```

## Mention Monitoring

```
Check my mentions every 30 minutes.
If someone asks a question, draft a reply and alert me.
```

```
Monitor my mentions for the keyword "help" or "question".
Prioritize those for response.
```

## Trend Monitoring

```
Check trending topics every hour.
If "AI" or "AI agents" is trending, alert me.
```

```
Monitor the hashtag #AIAgents from the last 24 hours.
Show me the most viral posts.
```

## Sentiment Monitoring

```
Search for posts mentioning "Hermes AI" from the last week.
Categorize them as positive, negative, or neutral.
Give me a sentiment summary.
```

```
Monitor posts about our product from the last month.
Track sentiment over time.
```

## List Monitoring

```
Show me the latest posts from my "AI Builders" list.
Summarize the key themes.
```

```
Monitor my "Competitors" list.
Alert me if any account posts about a product launch.
```
