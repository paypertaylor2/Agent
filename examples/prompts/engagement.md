# Engagement Prompts

Example prompts for engaging with others on X through Hermes.

## Liking

```
Like the top 5 posts in my timeline from the last hour.
```

```
Like all posts about "AI agents" from my mentions today.
```

```
Like the last 3 posts from @nousresearch.
```

## Replying

```
Reply to post 2047107136023650625 with "Great point! I've been thinking about this too."
```

```
Find the most engaging post in my mentions and draft a thoughtful reply.
```

```
Reply to @username's latest post with a relevant question.
```

## Following

```
Follow @nousresearch, @xai, and @hermes_agent.
```

```
Follow everyone who liked post 2047107136023650625.
```

```
Find the top 10 most engaging accounts posting about AI agents and follow them.
```

## Bookmarking

```
Bookmark all posts about "Hermes" from my timeline today.
```

```
Bookmark post 2047107136023650625 and add it to my "AI Research" list.
```

## Batch Engagement

```
Go through my timeline from the last hour:
- Like posts about AI
- Bookmark posts with threads
- Reply to any questions
```

```
Check my mentions from today:
- Like all mentions
- Reply to questions
- Bookmark important ones
```

## Strategic Engagement

```
Find the top 10 posts about "AI agents" from the last 24 hours.
Like them and follow the authors.
Draft a comment on the most popular one.
```

```
Search for people asking questions about AI agents.
Draft helpful replies to the top 5.
Ask me to approve before posting.
```
