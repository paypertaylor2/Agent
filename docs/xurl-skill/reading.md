# Reading

Read and search content on X through Hermes.

## Your Timeline

```
show my latest timeline
```

Returns a summarized, readable list of recent posts from accounts you follow.

```
show my last 50 timeline posts
```

## Search

```
search for posts about Hermes AI
```

```
search for "AI agents" from the last 24 hours
```

```
search for posts from @elonmusk about AI
```

```
search for posts with #AI and #crypto
```

## User Profiles

```
look up @elonmusk
```

Returns profile info: bio, follower count, recent posts, verification status.

```
show @nousresearch's last 10 posts
```

## Trends

```
show trending topics
```

```
show trending topics in the US
```

## Threads

```
get the full thread for post 2047107136023650625
```

Returns the entire conversation thread in readable format.

## Bookmarks

```
get all of my bookmarks
```

```
get my last 50 bookmarks
```

Returns bookmarked posts with metrics, media types, and usernames.

## Mentions

```
show my recent mentions
```


## Likes

```
show posts I've liked recently
```

## Output Format

Hermes presents results in a summarized, readable format — not raw JSON. Each result includes:
- Username and display name
- Post text
- Engagement metrics (likes, reposts, replies)
- Media type indicators
- Post link
