# Media

Post images, video, and GIFs through Hermes.

## Post an Image

```
post this image with the caption "Sunset from the office"
```

```
post /path/to/image.png with caption "Check this out"
```

## Post a Video

```
post /path/to/video.mp4 with caption "Demo of Hermes + xurl"
```

## Post a GIF

```
post /path/to/animation.gif
```

## Supported Formats

| Type | Formats | Max Size |
|------|---------|----------|
| Image | PNG, JPG, WEBP, GIF | 5MB |
| Video | MP4, MOV | 512MB |
| GIF | GIF | 15MB |

## Media + Text

```
post image /path/to/photo.jpg with caption "Building AI agents is fun #AI #Hermes"
```

## Multiple Media

```
post these 4 images with the caption "Thread of screenshots"
```

X supports up to 4 images per post.

## Alt Text

```
add alt text "Screenshot of Hermes terminal" to the image
```

## Media from URL

```
post the image from https://example.com/photo.jpg
```

## Tips

- Compress large images before posting
- Keep videos under 2:20 for best engagement
- Use alt text for accessibility
- GIFs autoplay in timeline
