# Interactions

Like, repost, bookmark, follow, and other social interactions.

## Like / Unlike

```
like post 2047107136023650625
```

```
unlike post 2047107136023650625
```

## Repost / Unrepost

```
repost 2047107136023650625
```

```
unrepost 2047107136023650625
```

## Bookmark / Unbookmark

```
bookmark post 2047107136023650625
```

```
unbookmark 2047107136023650625
```

## Follow / Unfollow

```
follow @nousresearch
```

```
unfollow @nousresearch
```

## Mute / Unmute

```
mute @username
```

```
unmute @username
```

## Block / Unblock

```
block @username
```

```
unblock @username
```

## Batch Interactions

```
like the last 5 posts in my timeline
```

```
bookmark all posts about AI from my mentions
```

```
follow everyone who liked post 2047107136023650625
```

## Safety

Hermes will ask for confirmation before:
- Blocking or unblocking accounts
- Batch operations affecting many posts/accounts
- Any destructive action (unfollow, unmute, etc.)
