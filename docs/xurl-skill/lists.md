# Lists

Manage X lists through Hermes.

## Create a List

```
create a list called "AI Builders" with description "People building AI agents"
```

```
create a private list called "Crypto News"
```

## Add to List

```
add @elonmusk to the "AI Builders" list
```

```
add @nousresearch, @xai to the "AI Builders" list
```

## Remove from List

```
remove @username from the "AI Builders" list
```

## View Lists

```
show all my lists
```

```
show the "AI Builders" list
```

```
show the last 50 posts from the "AI Builders" list
```

## Delete a List

```
delete the "Crypto News" list
```

## List Types

| Type | Visibility |
|------|------------|
| Public | Anyone can see and subscribe |
| Private | Only you can see |

## Use Cases

- **Curated feeds** — Group accounts by topic for focused reading
- **Monitoring** — Track competitors, influencers, or keywords
- **Research** — Organize accounts for analysis
- **Engagement** — Keep track of accounts you want to interact with
