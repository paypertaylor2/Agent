# Direct Messages

Send and manage direct messages through Hermes.

## Send a DM

```
send a DM to @username saying "Hey, loved your recent post!"
```

```
DM @username "Are you free to chat about the project?"
```

## Read DMs

```
show my recent DMs
```

```
show my DMs from @username
```

```
show the last 20 messages in my DM inbox
```

## Delete a DM

```
delete the DM with ID 123456789
```

## Conversation Thread

```
show the full conversation with @username
```

## Group DMs

```
send a DM to @user1 and @user2 saying "Let's sync up"
```

## Safety Notes

- Hermes will show you the message before sending
- DMs are private — Hermes stores conversation history locally
- Be mindful of rate limits on DMs
- Some accounts may not accept DMs from non-followers
