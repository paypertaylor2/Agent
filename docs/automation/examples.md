# Example Recipes

Real-world automation recipes for Hermes + xurl.

## Daily AI News Digest

**Goal:** Post a daily thread summarizing AI news from X.

```
Every morning at 9 AM:
1. Search X for "AI news" and "artificial intelligence" from the last 24 hours
2. Filter for posts with 50+ likes
3. Summarize the top 5 stories
4. Post a thread: "🤖 AI News Digest - [Date]"
5. Each tweet covers one story with a link to the original post
```

## Engagement Bot

**Goal:** Automatically engage with posts about topics you care about.

```
Every 2 hours:
1. Search for posts about "AI agents" and "terminal AI"
2. Like the top 3 most engaging posts
3. If any post has a question, draft a helpful reply
4. Ask me to approve before posting replies
```

## Weekly Analytics

**Goal:** Track your X performance weekly.

```
Every Monday at 10 AM:
1. Get all my posts from the last 7 days
2. Calculate total likes, reposts, replies, impressions
3. Identify the top 3 performing posts
4. Identify patterns (what topics, formats, times work best)
5. Send me a summary report
```

## Bookmark to Thread

**Goal:** Turn weekly bookmarks into a curated thread.

```
Every Sunday at 6 PM:
1. Get all bookmarks from the last 7 days
2. Categorize by topic
3. Select the best from each category
4. Post a "Weekly Favorites" thread with links
```

## Mention Responder

**Goal:** Never miss a question or important mention.

```
Every 30 minutes:
1. Check mentions from the last 30 minutes
2. If someone asked a question, draft a reply
3. If someone mentioned Hermes, draft a thank-you
4. Alert me with the drafts for approval
```

## Competitor Monitor

**Goal:** Track what competitors or key accounts are posting.

```
Every 6 hours:
1. Get the last 10 posts from @competitor1, @competitor2
2. Summarize their key messages
3. Note any product launches or announcements
4. Save to a "Competitor Intel" list
```

## Content Scheduler

**Goal:** Queue up a week of content in one session.

```
Once:
1. I provide 10 post ideas
2. Hermes drafts all 10 posts
3. I review and edit
4. Schedule 2 posts per day for the week
5. Confirm before each post goes live
```

## Follower Growth

**Goal:** Grow your network strategically.

```
Daily at 2 PM:
1. Search for posts about "AI agents" with 100+ likes
2. Identify the authors
3. Check if they're not already followed
4. Follow the top 5 most relevant accounts
5. Like their most recent post
```

## Thread Reader

**Goal:** Never miss a good thread.

```
Every 4 hours:
1. Search for "thread" + "AI" from the last 4 hours
2. Filter for threads with 200+ likes
3. Extract the key points
4. Summarize in a readable format
5. Bookmark the original thread
```
