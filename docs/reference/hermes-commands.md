# Hermes Commands

All available Hermes commands and shortcuts.

## Chat Commands

| Command | Action |
|---------|--------|
| `/help` | Show available commands |
| `/xurl` | Load the xurl skill |
| `/status` | Check Hermes status |
| `/model` | View or change model |
| `/reset` | Reset conversation |
| `/clear` | Clear chat history |
| `/history` | View conversation history |
| `/tools` | List available tools |
| `/config` | View current config |
| `/exit` or `/quit` | Exit Hermes |

## CLI Commands

| Command | Action |
|---------|--------|
| `hermes` | Start interactive chat |
| `hermes setup` | Run setup wizard |
| `hermes setup model` | Change model/provider |
| `hermes setup gateway` | Configure messaging |
| `hermes setup tools` | Configure tool providers |
| `hermes model` | Switch model or provider |
| `hermes config` | View current settings |
| `hermes config edit` | Open config in editor |
| `hermes doctor` | Run diagnostics |
| `hermes --version` | Show version |

## Auth Commands

| Command | Action |
|---------|--------|
| `hermes auth add xai-oauth` | Add xAI OAuth |
| `hermes auth list` | List stored credentials |
| `hermes auth logout xai-oauth` | Remove xAI OAuth |

## Cron Commands

| Command | Action |
|---------|--------|
| `hermes cron list` | List all cron jobs |
| `hermes cron add` | Add a new cron job |
| `hermes cron remove <name>` | Remove a cron job |
| `hermes cron enable <name>` | Enable a cron job |
| `hermes cron disable <name>` | Disable a cron job |
| `hermes cron run <name>` | Run a cron job now |

## Workflow Commands

| Command | Action |
|---------|--------|
| `hermes workflow list` | List workflows |
| `hermes workflow run <name>` | Run a workflow |
| `hermes workflow import <file>` | Import a workflow |
| `hermes workflow export <name>` | Export a workflow |

## Keyboard Shortcuts (Chat)

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line |
| `Ctrl+C` | Cancel current operation |
| `Ctrl+D` | Exit |
| `Ctrl+L` | Clear screen |
| `Up/Down` | Navigate history |
